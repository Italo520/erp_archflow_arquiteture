# Arquivo de Requisitos Arquivados - Milestone 1 (v1.0)

Este documento registra a validação e entrega dos requisitos de produto mapeados para a **Milestone 1 (Fase 0)**, arquivada em 28 de Maio de 2026.

*   **[x] REQ-003**: Criação de `.env.example` contendo chaves para banco, autenticação NextAuth e URL local do app.
*   **[x] REQ-004**: Garantir o padrão Singleton para o Prisma Client em `lib/prisma.ts`.
*   **[x] REQ-005**: Limpeza e saneamento do repositório, configurando o `.gitignore` para ignorar dados temporários de container, builds `.next` e logs.
