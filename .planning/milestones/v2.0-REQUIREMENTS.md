# Arquivo de Requisitos Arquivados - Milestone 2 (v2.0)

Este documento registra a validação e entrega dos requisitos de produto mapeados para a **Milestone 2 (Fase 1 - Parte A)**, arquivada em 28 de Maio de 2026.

## Requisitos de Projetos Arquitetônicos (PROJ)
*   **[x] REQ-015**: Inclusão de dados específicos de arquitetura no modelo `Project` do Prisma (estilo arquitetônico, tipo de construção, área total, número de andares, etc.).
*   **[x] REQ-016**: Implementação de Server Actions de CRUD completo para projetos, com revalidações de rotas e atualização de progresso.
*   **[x] REQ-017**: Recurso de duplicação de projeto para clonar a estrutura de fases e entregáveis como template.

## Requisitos de Estágios e Kanban (KANBAN)
*   **[x] REQ-018**: Ajustar estruturalmente o banco via migração do Prisma para adicionar chave estrangeira `projectId` na tabela `ProjectKanbanColumn`.
*   **[x] REQ-019**: Painel Kanban por projeto permitindo movimentação de cards com persistência em tempo de execução das colunas e ordenação.
